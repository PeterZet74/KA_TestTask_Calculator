Тестовое задание по курсу Java-разработчик
Задание:
Создать консольное приложение "КАЛЬКУЛЯТОР"
"КАЛЬКУЛЯТОР" должен выполнить арифметические действия
над часлами из введенной сроки.

Дополнительные условия:
- операции выполняются над целыми Арабскими и Римскими
  сислами в диапозоне от 0 до 10
- желаемое действие передается одной строкой в виде "А + Б"
- операнды должны быть одной системы, иначе выход по
  Исключению "Неверные данные"
- если указана неверная оперция (не +, -, *, /),
  выхоб по Исключению "Неверная операция"
- при делении на 0 (ноль) - Исключение "Деление на ноль"
- при вычислении Риских чисел - результат < (меньше) 1 -
  исключение "Недопустимый результат"

Решение:
1. Пуличить данные от пользователя с консольного ввода;
2. Проверить на соответствия:
 - Есть Римские числа?
 - Есть Арабские числа?
 - Есть Операция?
3. Первая проверка:
 - Арабские "И" Римские ЕСТЬ одновременно => Исключение "Неверные данные"
 - НЕТ Операции => Исключение "Неверная Операция"
4. Вычисляеи Реузльтат и Второй блок проверок:
 - Выделяем Операнды используя разделителем Операцию
 - Проверяем количество Операндов:
 -- < 2 => Исключение "Мало данныех"
 -- > 2 => Исключение "Много данных"
 - ЕСТЬ Римские => конвертируем Операнды РИМ_ту_АРАБ
 - Проверяем Операнд Б равен 0 "И" Операция "/" => Исключение "Деление на ноль"
 - Выполняем Операцию над Операднами А и Б => Результат
 - Проверяем: ЕСТЬ Рисмские и Результат > 1 => Исключение "Недопустимый результат"
 - Есть Римиские => конвертируем Результат АРАБ_ту_РИМ
 5. Выводим результат.

Вспомогательные данные и функции:
1. Перечисление соответствия Римских чисел Арабским:
- I = 1
- V = 5
- X = 10
- L = 50
- C = 100
- D = 500
- M = 1000
2. Перевод Римского числа в Арабское
- Перевод числа будет производиться проходом с лева на право с учетом вычитающей натации:
-- Если ЕСТЬ следующий символ и его значение БОЛЬШЕ текущего => вычитаем значение текушего из РЕЗУЛЬТАТА
-- If ( i < a.lehght ) { result =+ ( enum a[i] )}
3. Перевод Арабскго числа в Римское
